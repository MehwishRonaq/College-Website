# Web Technologies Coursework

This repository contains the coursework for the **Web Technologies** course (6th Semester). The primary focus is a modern, responsive website for **Government Post Graduate College (GPGC) For Women Mardan**, alongside various laboratory exercises.

## 🎓 Main Project: GPGC Women Mardan Website

A premium, academic-focused website designed to provide a modern interface for students and faculty. The design uses a sophisticated **Navy and Gold** color palette, reflecting the prestige of the institution.

### ✨ Key Features
- **Dynamic Hero Section**: High-quality autoplaying video header with overlay text.
- **Interactive Gallery**: A swipeable/scrollable gallery showcasing campus life.
- **Admission Portal**: A fully styled functional application form for prospective students.
- **Responsive Design**: Optimized for all devices, from desktops to mobile phones.
- **Premium Typography**: Integration of *Playfair Display* for headings and *Source Sans 3* for body text via Google Fonts.
- **Modern Animations**: Subtle scroll reveal effects for a smooth user experience.

### 🛠️ Technologies Used
- **HTML5**: Semantic structure for better accessibility and SEO.
- **CSS3**: Custom styling with advanced layouts (Flexbox, Grid) and animations.
- **JavaScript**: Client-side validation and interactive elements.

### 📂 Project Structure
```text
.
├── College-Website/           # Main Project Folder
│   ├── index.html            # Home page
│   ├── admission.html        # Admission application form
│   ├── contact.html          # Contact and location information
│   ├── style.css             # Main stylesheet
│   ├── logo.webp             # College branding
│   ├── video.mp4             # Hero background video
│   └── [Images].jpeg         # Gallery and asset images
├── lab1.html                 # Basic HTML tags, lists, and tables exercise
└── README.md                 # Project documentation
```

### 🚀 How to Run
1. Clone the repository or download the files.
2. Navigate to the `College-Website` directory.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.).
